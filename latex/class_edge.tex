\doxysubsection{Detailed Description}
La classe \mbox{\hyperlink{class_edge}{Edge}} représente une arête dans un graphe. 

Elle est dérivée de QGraphics\+Item et est responsable de dessiner une arête entre deux nœuds dans un graphe. Elle stocke les informations sur les nœuds source et destination, la couleur et la taille de l\textquotesingle{}arête. La méthode \mbox{\hyperlink{class_edge_ab554a765fd7a57fcdf289aa51b4df328}{adjust()}} est utilisée pour ajuster la position des nœuds source et destination sur l\textquotesingle{}arête. La méthode \mbox{\hyperlink{class_edge_a5ac25b0482fff6c68924031695fe4da7}{paint()}} est utilisée pour dessiner l\textquotesingle{}arête en utilisant les informations stockées.
