Detailed Description

La classe Edge représente une arête dans un graphe.

Elle est dérivée de QGraphicsItem et est responsable de dessiner une arête entre deux nœuds dans un graphe. Elle stocke les informations sur les nœuds source et destination, la couleur et la taille de l'arête. La méthode adjust() est utilisée pour ajuster la position des nœuds source et destination sur l'arête. La méthode paint() est utilisée pour dessiner l'arête en utilisant les informations stockées.
